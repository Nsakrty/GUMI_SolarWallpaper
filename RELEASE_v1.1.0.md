# GUMI_SolarWallpaper v1.1.0

## 🚀 New Features

### Enhanced Wallpaper System
- **5 wallpaper zones** (instead of 4):
  - Night (< -12°)
  - Sunrise/Sunset (-12° to 10°)
  - Morning/Evening (10° to 35°)
  - Daytime (35° to 50°)
  - Noon (50° to 90°)
- **Descriptive wallpaper filenames**:
  - night.JPG
  - sunrise_sunset.JPG
  - morning.JPG
  - day.JPG
  - noon.JPG

### Configuration Improvements
- **Configurable thresholds** in config.ini
- **Smart default values** if config is missing
- **Real-time config reloading** via tray menu

### User Experience
- **Next switch time prediction**
- **Enhanced tray tooltip**:
  - Solar altitude with 2 decimal precision
  - Current wallpaper status
  - Next scheduled switch time
